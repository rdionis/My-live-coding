# Recipe app

HTML,CSS and JavaScript with built-in sass support.

## Get Started

- [Setup](#setup)
- [Project Structure](#project-structure)

## Setup

1. clone this repo to your machine

1) Install the dependencies

   ```
   npm install
   ```

1) Run **live-server**

```
npm start
```
